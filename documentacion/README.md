# 🛒 ColorFly

ColorFly es una herramienta web interactiva diseñada para generar paletas de colores aleatorias con un solo clic
## 🔗 Enlaces
- **Demo en vivo:  https://joaco2003.github.io/ProyectoM1__JoaquinSuarez/
- **Repositorio:  (https://github.com/joaco2003/ProyectoM1__JoaquinSuarez)

## 🚀 Funcionalidades Principales
- Navegación fluida entre diferentes categorías de productos.
- Vista detallada de cada ítem con descripciones y precios.
- Interfaz completamente responsiva (adaptable a móviles y escritorio).

## 🛠 Tecnologías Utilizadas
- **Frontend:** React, React Router, HTML5, CSS3.
- **Herramientas:** Vite, Git, GitHub.
